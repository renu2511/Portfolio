The Project Idea

Safeable is a multi-channel application, telematics as one of the channels. It adjusts to preferred controls and connects to your smart watch and Syncs with itinerary to display and do what’s relevant, sends delay messages, sends notifications, one touch functions.

User Testing, Findings, Conclusion

We conducted user testing to determine the usability of our application. We asked the participants to complete a series of tasks for the mobile interactions and for the smart watch interactions. Participants were instructed to think aloud when completing the tasks. After completion, participants were asked for suggestions for improving the application.

All of the users were able to complete all of the tasks presented to them with minimal difficulty. Based on the users’ feedback the following changes were made to the first prototype:



Mobile

Automatically send message after selecting the message.

Add option to create new message

Add names to icons on the menu

Add title to the custom message screen

Add title to the custom message screen

Use icons on the navigation screen
